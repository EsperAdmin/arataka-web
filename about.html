About - arataka.dev

Arataka.dev is my website. I use Arataka as a pseudonym online, based off the character Reigen Arataka from the Anime Mob Psycho 100.


This website is a repository for things I may make like spiritsandsuch.lol, a re-creation of a defunct Spirits and Such Consulting website from the show Mob Psycho 100, and things I host like esper.lol, my generalized mastodon instance based on Mob Psycho 100.


I just re-started my programming journey and want to have this here as a place to centralize everything I have worked on and host. You can probably tell I like the Anime Mob Psycho 100 quite a bit. It really is a good one.


For now I really don't have much else to put here, so if you wanna check out the code for this simple but jank website you can visit my GitHub below.
